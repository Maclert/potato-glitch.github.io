About this site

This website contains data and basic guides for Last Cloudia, a mobile RPG available for Android and iOS.
Most of these guides are manually imported from spreadsheets that I authored or co-authored.

Use this website if you cannot access Google Sheets.

This website was made because I wanted to learn Bootstrap.
Modern frameworks are great! I love copy and paste templates!

Ongoing projects for Last Cloudia

  • LootTable Query – A spreadsheet that translates Japanese to Global item names/stage maps and has custom search functions for searching by item, stage, and crystal drops.
    • The drop guides are mirrored on this website.

  • Ark Ether Guide – A basic guide to help players figure out which Ark they should use their Etherions (or auto-run) on.

    • The guide is now hosted on this website. Future updates will appear on this website.
  • Unit Builder, Time Management and Story Reader – A catch-it-all bag of tools.
    • Plan your Ark levels, skill learning and unit builds
    • Keep track of your Ark collection
    • Search for Arks, skills, or equipment passives
    • Read the latest Ark and Unit lore, which has been edited (or rewritten) so that it isn't an obvious Google Translate job by Intern-kun

Our websites are based on two very different datasets. Vellde.org is based on Lake's spreadsheet which is great and has a lot of very useful comments. This website depends on the normalised dataset that Throne and I compiled for the unit builder.

One of our priorities is (and you can see this if you look at the Hidden tabs on the Unit builder spreadsheet) that all data is atomised and query-able. Ergo, SQL and text manipulation is doable, and HTML tables are easily compiled using our data. You can see the results in the Ark ether guide, which was compiled using modified arrays of the Equipment search functions. Once the tables are exported, all that's left is a cleanup job of grep/search and replace.
